AUTH SANCTUM (AGAR TIDAK SEMUA ORANG BISA MASUK KE API KITA)
1. Header Key : Accept, value : application/json
2. Body, Key : email  dan password pilih x www-form-urlencoded
3. Method POST, baseUrl api/login

==========
LOGOUT
1. URL yang digunakan {{baseUrl}}api/logout method yang digunakan juga maasih sama POST
2. Sebelumnya kita harus login dulu baru bisa masuk ke logut biar dapat token
3. pilih auth > Bearer > masukkan token yang sudah didapat send

================
REGISTER
1. perhatikan route api ini untuk mengetahui siapa saja yang bisa mengakses code tanpa token

    akses quote dan logout harus pakai token
    Route::middleware('auth:sanctum')->group(function () {
        Route::apiResource('/quote', QuoteController::class);
        Route::post('/logout', [ApiAuthController::class, 'logout']);
    });

    kalau register dan login tidak memakai token
    // buat register dan login
    Route::post('/register', [ApiAuthController::class, 'register']);
    Route::post('/login', [ApiAuthController::class, 'login']);

    create read update dan delete quote harus pakai token bearer
